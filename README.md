# 🎓 Academic Student Management System (ASMS)

A modern web-based Academic Student Management System developed using PHP Native, MySQL, Bootstrap 5, Chart.js, DataTables, and SweetAlert2.

## ✨ Features

- Dashboard Analytics
- Student Management (CRUD)
- Search & Pagination
- Export Excel
- Export PDF
- Print
- Responsive Design
- SweetAlert2 Notifications
- Chart.js Statistics

## 🛠 Tech Stack

- PHP Native
- MySQL
- Bootstrap 5
- Chart.js
- DataTables
- SweetAlert2

## 🚀 Installation

1. Clone this repository.
2. Import the database into MySQL.
3. Configure `config/db.php`.
4. Start Apache & MySQL using XAMPP.
5. Open `http://localhost/asms`.
